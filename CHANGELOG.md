# Changelog

## 0.9.1-alpha.4.1

- Added three large camera-view panels to the Live page with one-tap Program switching.
- Added Worship 1, Worship 2, and Worship 3 cues with distinct lighting and camera looks.
- Added three editable worship production looks.
- Removed Take Next, Back, and Hold controls from the Live page.
- Fixed page scrolling so content is no longer hidden behind the bottom navigation bar.

## 0.9.0-alpha.4

- Added explicit application and state schema versions.
- Added versioned migration support for Alpha 3 data.
- Added richer camera device records for future hardware integration.
- Added a shared camera preset library available to all cameras.
- Added per-camera preset assignment storage for future VISCA integration.
- Preserved the Alpha 3 `presetNames` interface during migration.
- Corrected Production Looks so applying a look recalls both cameras and their presets.
